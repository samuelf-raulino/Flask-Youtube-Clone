# Instalação

- Fazer download dos arquivos
    - No terminal na pasta escolhida pelo usuario
    - ``` git clone https://github.com/samuelf-raulino/Flask-Youtube-Clone.git ```

- Instalar Python

    - Fazer download e instalação em ``` https://www.python.org/downloads/ ```

- Adicionar bibliotecas
    - No terminal na pasta escolhida pelo usuario
    - ``` pip install -r requirements.txt ```

- Rodar site 
    - No Terminal dentro da pasta do sistema
    - ``` py app.py ```

# Resolução de erros

- Python não instalado
    - Reiniciar pc
- Python não funciona
    - Ao invés de usar py usar python
    - Desinstalar python e instalar de novo
- PIP não funciona
    - Use " py -m " ou " python -m " antes de qualquer comando pip
    - Ou desinstale o python e instale novamente
